About

TF2BP is different from every TF2 backpack viewer you know.

It loads all data Just-in-Time, directly from your browser, so every time you get fresh data about the player backpack you want see.

Of course there's a downside: since we don't use caches, it may take a while to download the items data and player data, but you won't care about this, because your backpack is always updated.

Another plus of TF2BP is that all code is open source (licensed under MIT License), so if you are a developer, you are free to contribute to make a better backpack (read below)

Contribute

All source code used on TF2BP is available on GitHub.

So, if you are a developer and think that any change (new features, bug fix, etc) should be made on TF2BP, you can fork our repository on GitHub.

If approved, your changes will be commited to the master branch, which is used on TF2BP.com
